# Detecting-Financial-Fraud-via-Deep-Graph-Representation-Learning-
This MSc Data Science project investigates traditional machine learning and graph representation learning for financial fraud detection using the PaySim dataset. It compares Logistic Regression, Random Forest, XGBoost, and Graph Convolutional Networks within a unified experimental framework. The study includes exploratory data analysis, class imbalance handling with SMOTE, hyperparameter optimization, feature importance analysis, graph construction, and graph neural network modeling. Model performance is evaluated using accuracy, precision, recall, F1-score, ROC analysis, and computational scalability. The results demonstrate the effectiveness of ensemble methods and provide a critical assessment of the opportunities and challenges of graph-based fraud detection in large-scale financial transaction networks.
